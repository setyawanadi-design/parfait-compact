## Additional Configuration

In addition to the original parfait theme for the firefox browser, the following settings are added for the horizontal tabs layout:

- `parfait.toolbar.separated` set to `true` : to enable a two-row separated layout for the URL bar and tab bar instead of the default unified layout.
- `parfait.toolbar.compact` set to `true` : to hide the tab bar (if `parfait.toolbar.separated` is also set to `true`) and only show the URL bar. (The tabs will smoothly slide out when hovering over the window top).
